# SD Fitness – Client

Vercel-ready React + Vite frontend for the SD Fitness gym website.

## Tech

- React 18
- React Router 6
- Tailwind CSS 3
- Redux Toolkit (+ React Redux)
- Framer Motion
- Axios

## Scripts

- `npm run dev` – start Vite dev server on port 5173
- `npm run build` – production build
- `npm run preview` – preview production build

Configure `VITE_API_BASE_URL` in `.env` to point at your Render (or local) API.
