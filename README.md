# SD Fitness

Premium modern full-stack MERN gym website with public marketing site and protected admin dashboard.

- Frontend: React (Vite), Tailwind CSS, Redux Toolkit, Framer Motion, React Router
- Backend: Node.js, Express.js, MongoDB Atlas, Cloudinary, JWT

This repo contains two apps:
- `client/` – Vercel-ready React frontend
- `server/` – Render-ready Express API

## Environment

Create `.env` files based on the examples:

- `server/.env.example` → `server/.env`
- `client/.env.example` → `client/.env`

## Development

From the repo root:

- `npm run dev:server` – start Express API (port 5000 by default)
- `npm run dev:client` – start Vite dev server (port 5173)

The frontend consumes the backend via `VITE_API_BASE_URL`.
